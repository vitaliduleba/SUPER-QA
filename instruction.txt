2. 	вводим почту, придуманный пароль, мини анкету
3. 	подтверждаем пароль ( сообщение на почте)
4.	создаем первый репозиторий (create a new repositiry)
5.	название репозитория, ввод
6.	появляется ссылка на наш репозиторий
6. 	cd /D 'путь к папке' -> в командной строке переходим к папке где будут храниться файлы через9папку создаем за ранее) 
7.	git clone 'ссылка на репозиторий'
8.	склонируется репозиторий с названием которое мы давали в пункте 5
9.	git test -> команда которая покажет репозитория

11.     git test -> пакажет что есть файл в паке но он не находится в репозитории
